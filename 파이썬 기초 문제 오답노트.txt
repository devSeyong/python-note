>> license_plate = "24가 2210"

정답
문자열에서 여러 글자를 가져오는 것을 슬라이싱이라고 부릅니다. 
음수 값은 문자열의 뒤에서부터 인덱싱 또는 슬라이싱함을 의미합니다. 
슬라이싱에서 시작 인덱스를 생락혀면 0으로 간주하고 끝 인덱스를 생략하면 문자열의 끝을 의미합니다.

print(license_plate[-4:])

	인덱스를 왼쪽(인덱스값이 양수)에서부터 보면 0시작이고
	오른쪽(인덱스값이 음수)에서보면 -1 시작입니다

* url 에 저장된 웹 페이지 주소에서 도메인을 출력하세요.
>> url = "http://sharebook.kr"
실행 예: kr

정답
문자열로 표현된 url에서 `.`을 기준으로 분리합니다. 분리된 url 중 마지막을 인덱싱하면 도메인만 출력할 수 있습니다.
url = "http://sharebook.kr"
url_split = url.split('.')
print(url_split[-1])
기본문법 - string.split(separator, maxsplit)
separator: 문자열을 나눌 기준이 되는 구분자입니다. (기본값은 공백)
maxsplit: 문자열을 나눌 최대 횟수입니다. (기본값은 -1, 즉 무제한으로 나누기)
split()은 문자열을 특정 구분자를 기준으로 나누는 함수입니다. 나눈 결과는 리스트 형태로 반환됩니다. 
주로 문자열을 여러 부분으로 나눠서 각 부분을 개별적으로 처리할 때 사용됩니다.

질문사항.
1번코드
string = 'abcdfeaaa'
string = string.replace('a', 'A')
print(string)

2번코드
string = 'abcd'
string.replace('b', 'B')
print(string)
1번코드랑 2번코드의 차이점이 뭔지 모르겠습니다.
1번코드는 결과값이 소문자 a가 대문자 A로 변환되고,
2번코드는 결과값이 소문자 b가 대문자 B로 변환이 안되는데,
둘다 문자열인데 1번코드는 변환이 되고 2번코드는 변환이 안됩니다..
